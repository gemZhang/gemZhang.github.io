Mac下go环境搭建
2018-01-26

作者：gemzhang，区块链开发者、架构师。

Go语言的优势：

  • Go语言编译的程序可以媲美C或C++代码的速度，语法简洁，而且更加安全、支持并行进程。这些优势让很多人青睐，更适合区块链的开发。

Mac OS X中golang的开发环境配置

  • 1.下载安装包
    官网地址：https://golang.org/dl/
    需要fq ,点击下载pkg包，下载完后双击安装。
  • 2检查
    在终端输入命令：go version
  • 3配置环境变量
    1.在终端输入命令：cd $HOME 切换到根目录
    2.执行命令：vim .bash_profile 打开文件
    3.点开始编辑，GOPATH:日常开发的根目录。GOBIN:是GOPATH下的bin目录。
    根据自己的情况输入，esc退出，：wq保存
    4.执行命令：source ~/.bash_profile 至此已经配置完成。
  • 4测试
    1.执行命令:vim text.go,然后输入

    2.执行命令：go run test.go，打印hello world,配置成功
