Mac下GitHub从零到一
2018-01-26

作者：gemzhang，区块链开发者、架构师。

一、准备并配置

  • 1.首先去GitHub官网注册账号

  • 2.配置本地全局变量
    git config –global user.email “you@example.com”
    you@example.com是GitHub注册邮箱
    git config –global user.name “Your Name”
    Your Name是GitHub名字

    二、项目应用

  • 1.点击右上角加号点击New repository创建仓库


编辑项目名称
– 项目描述
– 权限：公开Public（谁都可以查看），私有Private（需要收费，仅限自己
– Initialize this repository with a README 添加README，可以添加使用说明让别人了解该项目
– .gitignore 添加忽略文件，根据编程语言筛选使用场景
– Licenses 许可证，版权保护

– Settings(重点) 需要配置本地公钥才能对项目进行读写
打开终端输入命令
ssh-keygen -t rsa -C “you@example.com”
you@example.com是GitHub注册邮箱
输入电脑密码
复制Your public key has been saved in 后边的地址，不加最后的点

然后执行命令
cat /Users/TIANLONGSHUMA/.ssh/id_rsa.pub

复制返回的字符串到GitHub的key里面
勾选Allow write access

然后添加key,到此配置就可以对该项目进行读写

  • 2.点击Clone or download，点击复制链接
  • 3.打开终端 输入 git clone https://github.com/gemzhang/blockchain_blog.git // 刚才复制的地址，然后会将项目复制到本地
  • 4.切换到当前项目目录下，进行增删改查
    touch text.txt
    vim text.txt
    然后编辑后保存
  • 5.提交修改
    git add * //全部添加
    git add text.txt //添加指定文件
    git commit -m “提交描述”存放到本地暂存区
    git status 查看提交状态
    git push -u origin master 提交到远程仓库
  • 6.现在可以去GitHub刷新一下，查看自己提交的内容
